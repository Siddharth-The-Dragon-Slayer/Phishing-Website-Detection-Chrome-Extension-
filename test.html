📋 How to Test

  1. Load the Extension:
    • Go to chrome://extensions/
    • Enable "Developer mode"
    • Click "Load unpacked" and select the extension folder
  2. Test Automatic Detection:
    • Click on the test links above
    • Look for warning banners at the top of pages
    • High-risk sites should show red warnings
    • Medium-risk sites should show orange warnings
  3. Check the Popup:
    • Click the 🛡️ extension icon in the toolbar
    • View detailed risk analysis
    • See specific risk factors detected
    • Check statistics (sites checked, threats blocked)
  4. Test Badge Updates:
    • Watch the extension icon badge change colors
    • Green ✓ = Safe site
    • Orange ! = Medium risk
    • Red ⚠ = High risk

🔍 What to Look For

  • ✅ Warning banners appear on risky sites
  • ✅ Popup shows accurate risk scores
  • ✅ Risk factors are listed correctly
  • ✅ Badge icon updates based on risk level
  • ✅ Statistics increment properly
  • ✅ Warnings can be dismissed
  • ✅ No errors in browser console

🐛 Troubleshooting

  • No warnings showing: Refresh the page after loading the extension
  • Popup not working: Make sure you're not on a chrome:// page
  • Console errors: Check Developer Tools (F12) for error messages
  • Extension not loading: Verify all files are present and no syntax errors
